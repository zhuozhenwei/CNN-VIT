# CNN-VIT

# 实验环境
Python 3.10.8, Pytorch 2.1.2, Cuda11.8

# 数据集下载

CIFAR10 https://www.cs.toronto.edu/~kriz/cifar.html

# 运行方式

在对应的参数文件下面运行命令`python train_test.py`即可
